# Algo test

Computes the smallest super string from a given array of input strings.

## Install

### Prerequisites

Requires node **version 12** or above

### Commands
```bash
yarn
```
or
```bash
npm install
```

## Run
```bash
node index.js <file1> <file2> ... <fileN>
```

## Tests
```bash
yarn test
```
or
```bash
npm test
```
